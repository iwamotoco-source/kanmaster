01このアプリの使い方

KAN MASTERは、1級管工事施工管理技士1次試験の過去5年分（令和3〜7年）を、忘却曲線に基づいた効率学習で攻略するツールです。

演習
PRACTICE MODE
時間制限なし。スキマ時間に気軽に。分からない問題はあとで復習。
模試
EXAM MODE
本番と同じ165分の制限時間。直前期の実力チェックに。
復習
REVIEW MODE
採点後に正解・不正解を色分け表示。間違いを瞬時に確認。

▶ 操作手順

① 年度を選ぶ → ② PDF問題文を見ながら番号をタップ → ③「採点」ボタン → ④ 結果で弱点を確認 → ⑤ 翌日に同じ年度を再演習

💡 PDFについて
各年度にはデフォルトのPDF URLが設定されています。別タブで確認できます。オフライン利用には「PDF管理」からローカルファイルをアップロードしてください。

02忘却曲線とは

19世紀の心理学者 ヘルマン・エビングハウス の研究により、人は学習後 1時間で約56%、1日後に67%、1週間後に77%を忘れることが分かっています。

しかし「忘れかけたタイミングで復習」を繰り返すことで、記憶は長期定着します。これが最も効率的な暗記法の核心です。

▶ 理想的な復習タイミング

回タイミング目安時間
1回目学習当日（寝る前）10分
2回目翌日5分
3回目1週間後3分
4回目2週間後2分
5回目1か月後1〜2分

03社会人のスキマ学習法

社会人受験者の最大の課題は「まとまった時間が取れない」こと。調査では現代人には平均1日69分のスキマ時間があります。

1級管工事の合格ラインは総合60%（36/60問）かつ応用能力50%以上。4割は落としていい試験です。完璧を目指さず、頻出パターンの確実な積み上げが最善策です。

時間タイプ内容
5分MICRO電車待ち・昼食後など。5問だけタップ。
15分MINI通勤・昼休み。15〜20問の集中復習。
45分WEEKEND週末に年度別の通し演習。模試モードで。
💡 アクティブリコールの効果
「読み返す」より「思い出す練習」の方が記憶定着率は約2倍。過去問を解くこと自体が最高の暗記法です。

04結果画面の読み方

指標意味目標
総合得点60問中の正答率60%以上
応用能力問題B No.22-29の正答率50%以上（足切り）
苦手TOP10過去履歴から自動集計直前の最優先復習リスト
💡「苦手TOP10」の活用
このアプリは各問題の正答数と挑戦数を自動記録します。直前1週間はTOP10を毎日確認するだけでも得点が伸びます。

05推奨スケジュール（6ヶ月）

▶ PHASE 01｜基礎期（6〜4か月前）

最新2年分（R7・R6）を演習モードで1周ずつ。1日10〜15問ペース。解いた当日夜と翌日に再演習。

▶ PHASE 02｜拡張期（4〜2か月前）

R5・R4・R3を追加。新規10問＋復習15〜20問の並行学習。応用能力問題は短サイクルで反復。

▶ PHASE 03｜弱点期（2〜1か月前）

新規問題はストップ。苦手TOP10を毎日確認。週末は模試モードで通し演習。

▶ PHASE 04｜仕上げ期（直前1か月）

スキマ学習でTOP10を回す。本番1週間前にR7通し。当日朝はTOP10を眺めるだけ。

⚠ 残り1〜2か月の場合
いきなりR7・R6を何度も繰り返す戦略に切り替えを。直近2年分を完璧にすれば合格圏に届きます。
